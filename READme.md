# Endpoints

API → https://mlbb-api.vercel.app/

|  |  | 
| --- | --- |
| Exibe todos os heróis | /heroes |
| Filtra heróis por nome | /heroes/{name} |
| Retorna o herói pelo id | /{id} |
| Retorna os heróis pela função  | /roles/{role} |
